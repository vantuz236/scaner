# VulnScanner

Простой и быстрый CLI-сканер уязвимостей на Python. Помогает не искать CVE вручную для каждой службы на сервере.

## Что делает:
1. Сканирует указанный IP/домен через Nmap
2. Получает точные версии программ регулярками.
3. Обращается напрямую в актуальное NVD NIST API и ищет CVE для этих версий.
4. Фильтрует мусор (например, не путает обычный Apache с Apache Tomcat) и сравнивает версии как числа, чтобы не пропускать старый софт под описанием "before X.X.X".
5. Выводит понятный топ критических уязвимостей с оценками CVSS.

## Запуск:
Зависимости: в системе должен быть установлен обычный `nmap`.


nmap: https://nmap.org/download
